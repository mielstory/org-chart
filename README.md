# 組織図ジェネレーター

名前・役職・上司を入力すると、組織図が自動でできあがるツールです。
株式会社ミエルストーリーが「組織体制の構築」案件で、お客さま自身に入力してもらうために使います。

## 公開URL

https://mielstory.github.io/org-chart/

## 使い方

1. 上のURLを開く
2. 表に「名前」「役職」を入力し、「上司」をプルダウンから選ぶと自動でつながります
3. 一番上の人は上司を「（トップ）」のままにします
4. 「画像で保存（PNG）」「印刷 / PDF」で書き出せます

## 特長

- **単独HTML1枚・外部送信なし・オフライン動作。** 入力したデータはブラウザの中だけで処理され、外部のサーバーには一切送信されません。組織情報のような機微なデータでも安心して使えます。
- **自動保存。** 同じブラウザなら、閉じても続きから入力できます。
- **同姓同名に強い。** 上司は名前ではなく内部IDで結ぶため、誤接続しません。

## ローカルで使う

`index.html` をブラウザで開くだけ（ダブルクリック）。インターネット接続も不要です。

## カスタマイズ

`index.html` 冒頭の `--accent`（メインカラー2色）と、ヘッダーのロゴ「M」を差し替えると、ブランドに合わせられます（該当箇所はコメントで明示しています）。
